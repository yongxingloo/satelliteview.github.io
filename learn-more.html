How to use satelliteview.

This tool helps you search public satellite imagery over a selected area, compare overpasses through time, and export the results as a timelapse or a ZIP of individual frames.

Back to tool

1

Select an area

Use the map to choose where you want to search.

Draw area Rectangle mode

Click once for the first corner and once again for the opposite corner.

Use map view Current extent

Use the map viewport directly as the search area if you already zoomed into the region you need.

2

Configure the search

The current version is optimized for Sentinel-2 scenes.

Date range When

Choose the time interval you want to study.

Cloud cover Filter

Lower values keep cleaner scenes, but may reduce the number of results.

Scene limit Count

Controls how many candidate overpasses are requested from the search API.

Sequence Balanced / Strict / Dense

Adjusts how aggressively the app filters scenes for spatial consistency.

3

Read the outputs

After searching, the app updates several connected views.

Timeline When scenes happened

Shows the distribution of overpasses across the selected time range.

Statistics Quick summary

Includes number of scenes, date range, revisit time, and average cloud cover.

Results Scene list

Select a scene to focus it on the map and preview it in the timelapse panel.

Timelapse Main preview

Play through the current sequence and inspect the selected frame in more detail.

4

Export options

The current app supports two download paths.

Download WebM Animation export

Creates a video from the currently loaded sequence using the order shown in the timelapse panel.

Download ZIP Frame export

Packages all renderable frames into a single ZIP file for further analysis or external processing.
